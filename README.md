# The Angels Tour

## Project Vision
The Angels Tour is dedicated to creating transformative journeys that empower individuals to confront the ecological crisis and contribute to sustainable solutions. Our vision is to inspire collective action through immersive experiences that resonate with the core values of environmental stewardship and community engagement.

## Journey Pathways
### Dreamer
Dreamers unlock their creativity and explore visionary ideas for a sustainable future. This pathway offers inspirational workshops and activities designed to empower participants to think critically about the world around them.

### Guardian
Guardians are the protectors of our planet. This journey pathway focuses on education and activism, providing the tools and knowledge needed to advocate for ecological well-being in communities and beyond.

### Pioneer
Pioneers are innovators who lead the charge toward ecological restoration. This pathway encourages practical applications of sustainable practices, equipping participants with hands-on experience in conservation and restoration efforts.

## Ecological Crisis Narrative
In light of the ongoing ecological crisis, The Angels Tour aims to confront the challenges posed by climate change, biodiversity loss, and unsustainable practices. Our program encourages dialogue and action to address these pressing issues and inspires hope for a healthier planet.

## Fund Transparency Mechanism
We are committed to transparency in our funding processes. A comprehensive report detailing the allocation of funds will be available to participants and stakeholders, ensuring accountability and trust in our initiatives.

## Running the Development Server
To run the development server, please follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/taipeiedyen-bit/The-Angels-Tour.git
   ```
2. Navigate into the project directory:
   ```bash
   cd The-Angels-Tour
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```

Now you can access the project locally at `http://localhost:3000`. Enjoy developing!
